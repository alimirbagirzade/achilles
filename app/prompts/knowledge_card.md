# Knowledge Card Talimatı

Makaleden YALNIZCA aşağıdaki JSON şemasında çıktı üret. Açıklama, ön söz veya kod bloğu işareti ekleme; sadece JSON döndür.

{
  "paper_id": "...",
  "title": "...",
  "year": "...",
  "domain": "...",
  "main_claim": "...",
  "methods": [],
  "datasets": [],
  "trading_relevance": "...",
  "limitations": [],
  "possible_strategy_hypotheses": [],
  "risk_warnings": [],
  "implementation_notes": []
}

possible_strategy_hypotheses alanı test edilebilir, somut ve ölçülebilir olmalı.
risk_warnings alanı overfit, veri sızıntısı ve maliyet risklerini içermeli.
