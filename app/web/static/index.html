Sistem & Disiplin

Katmanlar

PDF → ingestion
   → SQLite + ChromaDB
   → RAG / bilgi kartı
   → (LoRA, dry-run)
   → Strateji IR → Backtest
   → Overfit → Evaluator

Disiplin Kuralları

  • Kaynak yoksa uydurma.
  • Test edilmeden "başarılı" deme.
  • Komisyon + slippage daima dahil.
  • Look-ahead yasak (shift+1).
  • In/out-of-sample bölmesi.
  • Kullanıcı girdisi asla çalıştırılmaz.

Güvenlik

  • Yalnız localhost'a bağlanır.
  • İsteğe bağlı token kimlik doğrulama.
  • Katı PDF doğrulama + boyut limiti.
  • CSP + güvenlik başlıkları.
  • IP başına hız sınırı.
  • Path-traversal koruması.
